# Glossary

## AdministrativeProcedure
Canonical root object representing one administrative service definition.

## Case
Runtime instance of a procedure execution for a specific applicant/request.

## Publishable
Record that satisfies Public API profile constraints and quality gates.

## Blocking quality condition
A quality issue that prevents publication until resolved.

## Non-blocking quality condition
A quality issue that may be published with warning under policy.

## Execution-ready workflow
Workflow that is non-empty, consistently ordered, semantically coherent, and publishable.

## Placeholder
Temporary value used during internal normalization (not allowed in public profile where prohibited).

## Internal Profile
Conformance profile for ingestion, normalization, and review workflows.

## Public API Profile
Conformance profile for external/public publication.

## Normative artifact
Schema or specification file that defines mandatory constraints.
